# todo-rs

Simple Interactive Terminal Todo App in Rust

## Quick Start

```console
$ cargo run
```

## Controls

| Keys                       | Description                                      |
| -------------------------- | ------------------------------------------------ |
| <kbd>z</kbd>, <kbd>s</kbd> | Move up and down                                 |
| <kbd>q</kbd>               | Quit                                             |
| <kbd>Enter</kbd>           | Perform tan action on the UI highlighted element |
